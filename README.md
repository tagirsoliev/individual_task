# Сервис для автоматизации процесса бронирования

## О проекте

В рамках производственной практики была разработана информационная система для предварительной записи клиентов к мастеру в сети парикмахерских. Данный проект представляет собой веб-приложение, позволяющее автоматизировать процесс управления заявками и значительно упростить взаимодействие между клиентами и администраторами салона.

Система разрабатывалась с учётом реальных бизнес-требований: удобство для конечного пользователя, надёжность хранения данных и простота администрирования.

## Цель разработки

Целью данной работы являлось создание современного веб-сервиса, который позволяет клиентам самостоятельно выбирать мастера и удобное время для посещения, а администраторам — своевременно обрабатывать поступающие заявки. Разработанное решение призвано заменить устаревшие способы записи (телефонные звонки, бумажные журналы) на удобный цифровой инструмент.

## Использованные технологии

В процессе разработки использовался современный технологический стек:

- **Backend** — NestJS (Node.js), реализующий REST API
- **Frontend** — React с TypeScript, обеспечивающий интерактивный пользовательский интерфейс
- **База данных** — PostgreSQL
- **ORM** — Prisma для работы с базой данных
- **Аутентификация** — JWT (JSON Web Tokens)
- **Контейнеризация** — Docker и Docker Compose

## Функциональные возможности

Система предоставляет следующие возможности:

**Для клиентов:**
- Регистрация и авторизация в системе
- Подача заявки на бронирование с выбором мастера, даты и времени
- Просмотр своих заявок и их текущего статуса
- Отмена заявки до её подтверждения

**Для администраторов:**
- Просмотр всех поступивших заявок
- Изменение статуса заявки (подтверждение или отклонение)
- Управление списком мастеров

## Запуск проекта

```bash
docker-compose up --build
```

После запуска приложение доступно по адресу: [http://localhost:5173](http://localhost:5173)

Данные для входа администратора:
- Email: `admin@barbershop.ru`
- Пароль: `admin123`
